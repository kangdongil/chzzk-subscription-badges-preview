# 언제나 치지직 구독배지

치지직(CHZZK) 구독 배지를 언제든지 한 번에 확인할 수 있도록 도와주는 Chrome 확장 프로그램입니다.

구독 중에는 현재 배지와 다음 배지만 보이는 기존 UI 대신,
“전체보기” 버튼을 통해 앞으로 받을 모든 구독 배지를 미리 확인할 수 있습니다.

---

## 주요 기능

- “전체보기” 버튼으로 전체 구독 배지 미리보기
- 라이브 / 영상 / 채널 페이지 지원
- 획득하지 않은 배지는 잠김 표시
- 실제 남은 기간 기준 구독 진행 게이지
- 다음 배지까지 남은 기간 숫자 표시
